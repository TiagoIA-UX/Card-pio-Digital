# Pasta "Cardápio Digital" – Documentação e referência

**O projeto ativo do Cardápio Digital não está aqui.**

O código em uso fica na **raiz do repositório** (pasta `01CardapioDigital`). Ali estão o Next.js, o `package.json` principal, as variáveis em `.env.local` e todo o desenvolvimento.

Esta pasta contém:

- **Cardapio_Digital/** – Cópia antiga/referência do projeto (não usar para desenvolvimento).
- **.obsidian/** – Notas e configuração do Obsidian.
- **Arquivos .md** – Documentação de produto, fluxos, marketing e objetivos.

Para trabalhar no projeto, abra a raiz do repositório e use os arquivos e scripts de lá.
